# -Challenge-Telecom-X-analisis-de-evasion-de-clientes-


## Descripción del Proyecto
Este proyecto tiene el objetivo de analizar una base de datos de clientes donde se aplicara técnicas de ETL (Extracción, Transformación y Carga) y realizar un Análisis Exploratorio de Datos (EDA) para descubrir los principales factores que motivan a los clientes a cancelar sus servicios (Churn).

##  Tecnologías Utilizadas
* **Lenguaje:** Python 3
* **Manipulación de Datos:** `pandas`, `json`
* **Visualización:** `matplotlib`, `seaborn`
* **Entorno:** Jupyter Notebook / Google Colab

##  Cómo ejecutar el proyecto
1. Asegúrate de tener instalado Python y las librerías requeridas. Puedes instalarlas ejecutando:
   `pip install pandas matplotlib seaborn`
2. Descarga el archivo de datos `TelecomX_Data.json` y colócalo en el mismo directorio que el notebook.
3. Abre el archivo principal del notebook (`TelecomX_LATAM.ipynb`).
4. Ejecuta las celdas de forma secuencial, desde la importación de librerías hasta la matriz de correlación.

##  Estructura del Análisis
1. **Extracción:** Carga de datos desde un archivo JSON.
2. **Transformación:** Desanidamiento de columnas complejas y limpieza de datos.
3. **Análisis Exploratorio:** Generación de gráficos descriptivos.
4. **Conclusiones:** Informe con los principales insights y recomendaciones de negocio.
